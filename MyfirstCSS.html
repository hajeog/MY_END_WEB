CSS 란?

CSS란 여러분들이 HTML로 구성되어있는 이 세상의 웹페이지에서 가장 아름답고 이쁜 도구들 입니다.
CSS가 없으면 해당 글도 꾸밈 없이 전달력이 부족하거나 이해하기 어려운 웹 페이지로 변할수도 있습니다.
만약 CSS가 없는 세상의 모습은 여기를 클릭하시면 됩니다.
해당 프로그램에 들어가있는 CSS를 보실려면 'CSS 코드 보기'를 클릭 을 하시면 대항 웹 페이지의 CSS 코드를 볼수 있습니다.

또한 개인적으로 깃허브를 보고 작성한 코드를 보고 싶으면...
여기를 클릭해 주시면 감사 합니다.
